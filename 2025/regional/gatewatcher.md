# Gatewatcher CTI (France)

## Overview
Gatewatcher is a French cybersecurity company providing network detection and response (NDR) solutions combined with threat intelligence capabilities. Specializing in advanced threat detection through deep packet inspection and behavioral analysis, Gatewatcher offers both network security appliances and threat intelligence services, with strong focus on European threats and regulatory compliance (GDPR, NIS2).

## Key Features
- Trackwatch NDR platform with integrated CTI
- Deep packet inspection and analysis
- Behavioral threat detection
- European threat landscape intelligence
- Malware analysis and tracking
- Network forensics capabilities
- Threat actor tracking
- Integration with European security ecosystem
- GDPR and NIS2 compliance features
- On-premise and cloud deployment
