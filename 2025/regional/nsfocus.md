# NSFOCUS (China)

## Overview
NSFOCUS (绿盟科技) is a leading Chinese cybersecurity company specializing in network security, threat detection, and threat intelligence. Founded in 2000, NSFOCUS provides comprehensive security solutions including DDoS protection, threat intelligence, and security operations platforms, primarily serving Chinese enterprises, carriers, and government agencies with strong expertise in network-level threats.

## Key Features
- NSFOCUS Threat Intelligence (NTI) platform
- DDoS attack intelligence and mitigation
- Network security threat detection
- Vulnerability intelligence and management
- APT threat intelligence
- Malware analysis and tracking
- Carrier-grade threat intelligence
- Industrial security intelligence
- Cloud security threat intelligence
- Integration with NSFOCUS security products
