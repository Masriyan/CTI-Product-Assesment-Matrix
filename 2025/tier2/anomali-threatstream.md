# Anomali ThreatStream

## Overview
Anomali ThreatStream is a purpose-built Threat Intelligence Platform (TIP) that aggregates, normalizes, and enriches threat intelligence from multiple sources. Founded in 2013, Anomali focuses exclusively on threat intelligence management and operationalization, making it one of the leading pure-play TIP vendors in the market.

## Key Features
- Threat intelligence aggregation from 200+ sources
- Automated indicator normalization and deduplication
- Threat intelligence scoring and prioritization
- MITRE ATT&CK framework integration
- Anomali Match for SIEM log correlation
- Anomali Lens for analyst-driven hunting
- Threat intelligence sharing and collaboration
- Playbook automation capabilities
- Graph-based relationship mapping
- Threat actor and campaign tracking

## Pros
- Purpose-built TIP with deep functionality
- Excellent threat intelligence aggregation and normalization
- Strong free community offering (STAXX) for basic TI feeds
- Good integration with major SIEM and security platforms
- Flexible deployment options (cloud, on-premise, hybrid)
- User-friendly analyst interface
- Strong community and intelligence sharing features
- Good MITRE ATT&CK integration
- Effective false positive reduction
- Active development and feature updates

## Cons
- Can be expensive for full-featured deployment
- Requires dedicated analyst resources to maximize value
- Learning curve for advanced features
- Some customers report performance issues with large datasets
- Advanced analytics features require premium licensing
- Third-party feed quality varies
- Limited SOAR capabilities compared to dedicated SOAR platforms

## Use Cases
- Central threat intelligence aggregation and management
- SIEM enrichment and IOC correlation
- Threat hunting operations
- Security operations center (SOC) intelligence operations
- Multi-source threat intelligence normalization
- Threat intelligence sharing within industry groups
- Incident response and forensics
- Vulnerability prioritization with threat context

## Integration Capabilities
- SIEM integrations (Splunk, Microsoft Sentinel, IBM QRadar, Chronicle)
- SOAR platforms (Palo Alto Cortex XSOAR, Splunk SOAR)
- Firewalls and network security (Palo Alto, Cisco, Fortinet, Check Point)
- EDR/XDR platforms (CrowdStrike, Microsoft Defender, SentinelOne)
- Ticketing (ServiceNow, Jira)
- MISP and OpenCTI integration
- RESTful API and Python SDK
- STIX/TAXII server and client
- 200+ pre-built integrations

## Pricing Model
- Subscription-based annual licensing
- Tiered pricing based on user count and features
- Per-analyst pricing model
- Additional costs for Anomali Match and Lens modules
- Free STAXX community edition for basic feeds
- Custom enterprise pricing
- Professional services available for deployment
- Pricing typically starts at $50,000+ annually

## Target Organization Size
- Mid-market to Enterprise
- Organizations with mature security operations
- Typically 1,000+ employees
- Best suited for organizations with dedicated threat intelligence teams
