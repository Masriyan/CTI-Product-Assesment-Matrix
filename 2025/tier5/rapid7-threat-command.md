# Rapid7 Threat Command (IntSights)

## Overview
Rapid7 Threat Command, formerly IntSights, is an external threat intelligence and digital risk protection platform now integrated into the Rapid7 portfolio. Threat Command provides organizations with visibility into threats from the clear, deep, and dark web, combining automated collection with expert analysis to deliver actionable intelligence about targeted threats, data leaks, and adversary activity.

## Key Features
- External threat intelligence platform
- Deep and dark web monitoring
- Targeted attack warnings
- Brand and executive protection
- Credential leak detection
- Threat actor tracking and profiling
- Phishing and domain monitoring
- Vulnerability and exploit intelligence
- IOC feeds and malware intelligence
- Integration with Rapid7 Insight platform

## Pros
- Strong integration with Rapid7 ecosystem (InsightVM, InsightIDR, etc.)
- Good dark web and underground monitoring
- Targeted attack intelligence capabilities
- User-friendly platform interface
- Automated threat prioritization and scoring
- Good value when bundled with Rapid7 products
- Strong customer support
- Regular threat intelligence reports
- Proactive analyst engagement
- Growing integration capabilities post-acquisition

## Cons
- Best value within Rapid7 ecosystem
- Post-acquisition integration with Rapid7 products ongoing
- Standalone offering less competitive than integrated
- Some customers report feature overlap with other Rapid7 products
- Integration with non-Rapid7 tools moderate
- Pricing can be complex with Rapid7 bundles
- API capabilities moderate

## Use Cases
- External threat intelligence monitoring
- Targeted attack early warning
- Brand and executive protection
- Credential compromise detection
- Vulnerability prioritization with threat context
- SOC threat intelligence operations
- Integration with Rapid7 vulnerability management
- Incident response and threat hunting

## Integration Capabilities
- Native integration with Rapid7 Insight platform (InsightVM, InsightIDR, InsightCloudSec)
- SIEM integrations (Splunk, Microsoft Sentinel, Chronicle)
- Threat intelligence platforms (Anomali, ThreatConnect, MISP)
- SOAR platforms (limited)
- Ticketing (ServiceNow, Jira)
- RESTful API
- STIX/TAXII support
- Webhook and email notifications

## Pricing Model
- Subscription-based annual licensing
- Bundled pricing with Rapid7 Insight platform
- Standalone Threat Command pricing available
- Tiered pricing based on features and organization size
- Custom enterprise pricing
- Typically $40,000-100,000+ annually
- Discounts for Rapid7 platform customers
- Multi-year commitments may reduce cost

## Target Organization Size
- Mid-market to Large Enterprise
- Organizations with 500+ employees
- Existing Rapid7 customers
- SOC and security operations teams
