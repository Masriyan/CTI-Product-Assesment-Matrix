# Microsoft Defender Threat Intelligence

## Overview
Microsoft Defender Threat Intelligence (formerly RiskIQ) provides comprehensive threat intelligence leveraging Microsoft's vast telemetry data, including signals from billions of endpoints, email messages, and cloud services. Integrated into the Microsoft Security ecosystem, it offers attack surface management, threat intelligence, and digital risk protection capabilities.

## Key Features
- Attack surface discovery and monitoring
- Threat intelligence from Microsoft's global sensor network
- Infrastructure analysis (WHOIS, DNS, SSL certificates)
- Adversary infrastructure tracking
- Vulnerability and exposure intelligence
- Reputation scoring for IPs, domains, and files
- Integration with Microsoft 365 Defender
- Digital risk protection and brand monitoring
- Threat analytics and correlation

## Pros
- Massive telemetry data from Microsoft ecosystem (Windows, Office, Azure)
- Deep integration with Microsoft Security products
- Excellent value for Microsoft-centric organizations
- Comprehensive attack surface visibility
- Strong infrastructure intelligence and passive DNS data
- Advanced threat analytics with AI/ML
- Included in certain Microsoft 365 E5 licenses
- Extensive threat actor profiling
- Good coverage of nation-state and cybercrime threats
- Continuous attack surface monitoring

## Cons
- Best suited for Microsoft-heavy environments
- Integration challenges with non-Microsoft security stacks
- UI can be complex for new users
- Some features require multiple Microsoft licenses
- May overlap with other Microsoft security products
- Limited dark web monitoring compared to specialized vendors
- Steep learning curve for advanced features
- Feature disparity between standalone and integrated offerings

## Use Cases
- Organizations heavily invested in Microsoft ecosystem
- Attack surface management and external exposure monitoring
- Microsoft 365 and Azure security enhancement
- Threat hunting in hybrid Windows/cloud environments
- Infrastructure intelligence and domain/IP reputation checks
- Supply chain and third-party risk assessment
- Brand protection and digital risk monitoring
- Vulnerability intelligence and patch prioritization

## Integration Capabilities
- Native integration with Microsoft 365 Defender
- Microsoft Sentinel (Azure SIEM) integration
- Microsoft Defender for Endpoint
- Microsoft Defender for Cloud Apps
- Azure Security Center integration
- Power BI for custom reporting
- RESTful API
- Limited third-party SIEM integration
- STIX/TAXII support
- Microsoft Graph API connectivity

## Pricing Model
- Included in Microsoft 365 E5 and E5 Security licenses
- Standalone subscription available
- Usage-based pricing for API queries
- Tiered pricing based on features and data volume
- Enterprise licensing with Microsoft Enterprise Agreement
- Additional costs for premium threat intelligence features
- Pricing typically bundled with broader Microsoft security suite

## Target Organization Size
- Mid-market to Enterprise
- Best suited for Microsoft-centric organizations
- Organizations with 500+ employees
- Particularly valuable for Azure and M365 customers
