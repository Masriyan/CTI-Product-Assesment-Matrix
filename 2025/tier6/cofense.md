# Cofense Intelligence (Phishing-focused)

## Overview
Cofense Intelligence (formerly PhishMe Intelligence) is a phishing-focused threat intelligence service that provides real-time intelligence on active phishing campaigns, malicious infrastructure, and email-based threats. Cofense specializes in crowdsourced intelligence from millions of email users and automated analysis of phishing threats, helping organizations detect and respond to email-borne attacks.

## Key Features
- Real-time phishing campaign intelligence
- Crowdsourced threat reporting network
- Malicious infrastructure tracking (C&C, phishing sites)
- Email threat IOCs and indicators
- Malware payload analysis
- BEC (Business Email Compromise) intelligence
- Credential phishing detection
- Integration with Cofense email security products
- Automated IOC feeds for blocking
- Active phishing campaign alerts

## Pros
- Leading expertise in phishing threat intelligence
- Crowdsourced intelligence from millions of users
- Real-time active phishing campaign detection
- Excellent integration with email security platforms
- High-fidelity IOCs for blocking malicious infrastructure
- Good value for email-focused security
- Fast alerting on active campaigns
- Strong malware analysis capabilities
- Reduces phishing incident response time
- Comprehensive coverage of email threats

## Cons
- Narrow focus on email/phishing threats
- Limited broader CTI capabilities
- Best value with Cofense email security products
- Integration outside email security platforms moderate
- Not suitable for comprehensive threat intelligence needs
- Pricing can be high for standalone intelligence
- Some customers report alert volume requires tuning

## Use Cases
- Email security operations
- Phishing threat detection and response
- SOC email threat analysis
- Incident response for phishing attacks
- Security awareness and training validation
- Malicious infrastructure blocking
- BEC detection and prevention
- Email gateway integration

## Integration Capabilities
- Native integration with Cofense email security products (Triage, Vision)
- Email gateways (Proofpoint, Mimecast, Microsoft EOP/Defender)
- SIEM integrations (Splunk, Microsoft Sentinel, QRadar)
- SOAR platforms (Palo Alto Cortex XSOAR, Splunk SOAR)
- Threat intelligence platforms (Anomali, ThreatConnect, MISP)
- EDR platforms (limited)
- RESTful API
- STIX/TAXII feeds

## Pricing Model
- Subscription-based annual licensing
- Pricing based on organization size (user count)
- Bundled with Cofense email security products
- Standalone intelligence subscription available
- Custom enterprise pricing
- Typically $20,000-80,000 annually
- Volume discounts for large user bases

## Target Organization Size
- Small to Large Enterprise
- Organizations with 500+ employees
- Email-heavy industries (finance, healthcare, professional services)
- SOC teams focused on email threats
