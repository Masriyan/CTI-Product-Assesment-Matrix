# Farsight Security (DNSDB)

## Overview
Farsight Security provides the world's largest passive DNS (pDNS) database and real-time DNS intelligence through its DNSDB platform. Specializing in DNS-based threat intelligence, Farsight helps security teams investigate malicious infrastructure, track threat actor domains, and understand DNS patterns associated with cybercrime and targeted attacks.

## Key Features
- DNSDB passive DNS database (largest in the world)
- Historical and real-time DNS intelligence
- DNS query and response monitoring
- Newly Observed Domains (NOD) detection
- Domain generation algorithm (DGA) detection
- DNS-based threat hunting capabilities
- API access for automated queries
- Integration with security analysis workflows
- DNS infrastructure mapping
- Threat actor infrastructure tracking

## Pros
- Largest and most comprehensive passive DNS database
- Excellent for DNS-based threat hunting and investigations
- Real-time newly observed domain detection
- Valuable for infrastructure analysis and attribution
- Strong API for integration and automation
- Trusted by law enforcement and intelligence agencies
- Long historical DNS data retention
- Good for DGA detection and analysis
- Reliable and high-quality data
- Essential tool for advanced threat research

## Cons

- Premium pricing for commercial access

- Requires DNS expertise to maximize value

- Limited context beyond DNS data
- Integration primarily through API vs. native
- Query limits on lower tiers
